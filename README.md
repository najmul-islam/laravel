## Setup

To get started with the projects in this repository, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/najmul-islam/laravel.git
   cd vue.js
   ```

2. Go any folder:

   ```bash
   php artisan serve
   ```
